# 2023-dn-NguyenPhanMinhThinh-demo_crawl
To run this project, you need to use cmd to install venv
## Initilize venv 
```cmd
python -m venv venv
```
## Active venv and install lib
1. Activate vnev
```cmd
venv\bin\activate.bat
```
2. Install lib
```cmd
pip install --upgrade pip
pip install -r requirements.txt
```
